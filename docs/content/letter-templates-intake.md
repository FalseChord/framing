# 信件內容補完 — 資料填寫檔

## 怎麼填

- 每個類別、每個方案貼「一份完整範例信」即可，**不需要自己寫 `{{變數}}` 語法** — 我會比對範例、自動拆出哪些是變數（個案代號、心理師姓名、日期等）、哪些段落是特定方案才有的差異。
- 個案代號、姓名、日期等請先換成假資料再貼（例如 `A999`、`王小明`、`2026/01/01`），**不要貼真實個案資訊**（本專案的個資/跨境處理原則見設計文件，貼真實資料進來等同送到雲端 AI 服務，違反當初自架的理由）。
- 有多種方案（一般／青壯／北捷／EAP）的類別，**每個方案都要各貼一份範例**，這樣我才能比對出共用段落 vs. 方案專屬段落。只有一種內容的類別（沒有方案差異的）就只貼一份。
- 標題（主旨）跟內文都要貼。
- 全部填完後告訴我一聲，我會一次讀取這個檔案，轉換成正式模板。還沒填的類別留白即可，我不會處理空白區塊，之後可以再補。
